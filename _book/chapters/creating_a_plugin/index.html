Netbeast Documentation

Netbeast plugins

A Netbeast plugin is a npm module that starts a service, thus, creating a functionality. It's the sauce behind Netbeast API engine. With a plugin you can register a resource for others to use, like a new brand of smart bulbs to be controlled as another light source.
